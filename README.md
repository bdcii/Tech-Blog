# Tech Blog
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description
        
*What does it do?*
        
   This is a Blog that allows users to sign up and create/edit posts, as well as create/edit comments on other users' posts.
             
*Technologies Used:*
        
    JavaScript, NPM and various dependencies(sequelize, express-session, handlebars, express), MySQL,
